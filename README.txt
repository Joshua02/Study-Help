

pip freeze to get requirements file

First time running the program, open the terminal in the folder with README.txt and run each of these commands:
1. python -m pip install --upgrade pip
2. py -m pip install --user virtualenv
3. py -m venv env
4. .\env\Scripts\activate
5. pip install -r requirements.txt
6. python website.py

Then afterwards you just have to open the terminal in the folder wiht README.txt:
1. .\env\Scripts\activate
2. python website.py
